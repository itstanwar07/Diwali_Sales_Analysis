# Diwali Sales Analysis using Python

## Overview

This project was completed as part of my Machine Learning & AI Internship. The objective was to analyze Diwali sales data to understand customer purchasing behavior and extract business insights that can support marketing strategies during festive seasons.

The project follows a complete Exploratory Data Analysis (EDA) workflow, including data cleaning, preprocessing, visualization, and insight generation using Python.

---

## Problem Statement

Analyze the Diwali sales dataset to identify customer purchasing patterns across:

- Gender
- Age Groups
- Marital Status
- Occupation
- States
- Zones
- Product Categories

The goal is to discover meaningful trends that can help businesses improve customer targeting and maximize festive sales.

## Key Insights

Some important observations from the analysis include:

- Female customers contributed significantly to overall sales.
- Customers aged 26–35 were among the highest purchasers.
- Married customers showed higher purchasing activity.
- Certain states generated substantially higher revenue.
- Specific occupations had stronger buying capacity.
- A few product categories dominated festive sales.
